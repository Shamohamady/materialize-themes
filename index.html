Materialize Themes

Pre-generated materialize css color combinations.

Materialize CSS is an awesome CSS framework for building material design websites.

Materialize themes has pre-generated all primary/secondary color combinations of the materialize css framework for use in your project!
No need to download/compile sass files!

Simply replace the link to materialize.css with a link to one of the themes. The current version used to generate the themes is 0.97.8
